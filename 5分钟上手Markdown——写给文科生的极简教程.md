# 📘 5 分钟上手 Markdown
## 写给文科生的极简教程
📍 读者定位：零编程基础、没听过"标记语言"这个词的纯文科生。

🎯 学习目标：花 5 分钟掌握 6 个最常用的 Markdown 语法，立刻上手写一篇格式漂亮的短文。

##  Markdown 是什么？一句话说清
你可以把 Markdown 理解成"带快捷键的纯文本"。

你在 Word 里加粗文字，需要选中文字然后点 B 按钮。但在 Markdown 里，你只需要在文字两边加上两个星号 ** 就行了。原理是一样的，只是操作方式不同——Markdown 用符号来控制格式，不需要鼠标点来点去。

那 Markdown 有什么好处呢？三个字：快、净、稳。

- 快：你打字的时候顺手加几个符号就行，手不用离开键盘去摸鼠标。

- 净：文件没有 Word 那种复杂的格式代码，就是纯纯的文字，打开速度快。

- 稳：任何电脑、任何系统都能打开，永远不会"排版乱掉"。

## 6 个核心语法
你只需要掌握下面 6 个，就能覆盖 90% 的日常写作场景。

# 语法 1：标题
你输入什么：

    # 一级标题

    ## 二级标题

    ### 三级标题
屏幕上会出现什么：

你输入的屏幕上的效果
# 一级标题
## 二级标题
### 三级标题
***💡 规律就是：几个 # 号就是几级标题，# 号和文字之间必须有一个空格。***

# 语法 2：加粗和斜体
你输入什么：

    **这是加粗的文字**
    *这是斜体的文字*
    ***这是加粗加斜体***

屏幕上会出现什么：

**这是加粗的文字**

*这是斜体的文字*

***这是加粗加斜体***

***💡 规律就是：两个 * 包起来 = 加粗，一个 * 包起来 = 斜体，三个 = 又粗又斜。***

# 语法 3：列表
无序列表（圆点符号）

你输入什么：

    - 苹果
    - 香蕉
    - 橘子

屏幕上会出现什么：

- 苹果
- 香蕉
- 橘子

***💡 规律就是：短横线 - 加空格，就生成一个圆点列表。***

有序列表（数字编号）

你输入什么：

    1. 第一步：打开电脑
    2. 第二步：打开 VS Code
    3. 第三步：新建文件

屏幕上会出现什么：

1. 第一步：打开电脑

2. 第二步：打开 VS Code

3. 第三步：新建文件

***💡 规律就是：数字加英文句号加空格，就生成自动编号的列表。***

# 语法 4：链接
你输入什么：

    [点击这里访问百度](https://www.baidu.com)

屏幕上会出现什么：

[点击这里访问百度](https://www.baidu.com)

***💡 规律就是：[显示的文字] 后面紧跟着 (网址)，中间没有空格。***

# 语法 5：代码（行内代码）

你输入什么：

请用 `print("hello")` 输出一句话。

屏幕上会出现什么：

请用 print("hello") 输出一句话。

***💡 规律就是：用反引号 `（键盘上 Esc 键下面那个键）把内容包起来，文字会变成等宽字体，像代码一样显示。***

# 语法 6：图片
你输入什么：

![这是图片的描述](https://example.com/photo.jpg)

屏幕上会出现什么：



一张图片（显示 photo.jpg 这张图，如果图片加载失败，会显示"这是图片的描述"这段文字）。

***💡 规律就是：! + [图片描述] + (图片网址)。和链接很像，只是前面多了个 !。***

# 🎯 实战小练习
现在你已经学完了全部 6 个语法，请花 5 分钟完成下面这个练习。

**任务：用 Markdown 写一段 200 字左右的自我介绍，要求必须包含：**

1. 一个一级标题（你的名字）

1. 至少一处加粗

1. 一个无序列表（比如你的兴趣爱好）

1. 一个链接（比如你的 GitHub 或微博主页）

**写完后的样子大概是：**

# 张三

我是 **英语专业** 的一名大四毕业生，正在学习技术文档写作。

我的兴趣爱好：
- 阅读
- 写作
- 摄影

欢迎访问我的个人主页：[点击这里](https://zhangsan.github.io)

完成之后：把这个文件保存为 intro.md，用 VS Code 打开预览看看效果 —— 你会很有成就感的！

# 🔧 常见报错排查（新手必看）
## 新手刚开始写 Markdown 最容易遇到这些问题，提前知道能省掉很多折腾时间。

### ❌ 问题 1：写了 #标题 但没有变成标题
原因：# 和标题文字之间没有空格。

修正：写成 # 标题（井号加空格加文字）。

### ❌ 问题 2：写了 **文字** 但没有加粗
原因：你用的是中文输入法下的星号 ＊（中文全角符号），不是英文输入法的 *（英文半角符号）。

修正：把输入法切换到英文状态，重新打 **。

### ❌ 问题 3：点链接显示 404 Not Found
原因：链接地址写错了，或者链接里包含了中文字符。

修正：

检查网址是否完整（必须以 https:// 或 http:// 开头）。

如果网址里有中文，复制粘贴的时候不要改动，保持原样。

### ❌ 问题 4：图片不显示，只显示一串文字
原因：图片地址填错了，或者那张图本身已经不存在了。

修正：

确认图片网址能直接在浏览器里打开。

如果是本地的图片（比如 photo.jpg），需要上传到网上图床（如 sm.ms）拿到网址再填进去。
